# Emotion Detection using Edge + Dual Stream Learning

## Idea
Can structural features (edges) improve emotion recognition?

This project explores combining real facial images with edge-based representations inspired by cartoon drawings.

## Approach
- Dual-stream CNN (MobileNetV2)
- Real images + edge inputs
- Feature fusion using concatenation

## Results
- Stage 1: ~66.7%
- Stage 2: ~74.0%
- Final Model: ~83.5%

## Key Insight
Edges capture structural information that improves emotion detection performance.
